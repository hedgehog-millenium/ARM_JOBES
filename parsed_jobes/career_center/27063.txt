  <p><b>DURATION:</b>&nbsp;&nbsp;Permanent</p>

  <p><b>JOB DESCRIPTION:</b>&nbsp;&nbsp;Working in Praemium, the incumbent's role will be to assist with the development and maintenance of financial services related to software products for the United Kingdom market. He/ she will also be required to enhance the Praemium suite of applications. This includes development on Windows Forms-based designs, Microsoft SQL database and data entity work, report designs as well as some web service tools and APIs. This can include displaying and editing data, constructing report output based on this data, manipulating the data and object models to use on advanced tools via a Wizard type interface. The incumbent will work with the local and global teams. He/ she will also support operational aspects of the technology platform and later shift to extending the application over time.</p>
  <p><b>REQUIRED QUALIFICATIONS:</b><br>
- At least 2 years of experience in software development;<br />
- Knowledge of C# and .NET;<br />
- Familiarity with Microsoft Development environment, including but not exclusive to Microsoft Visual Studio;<br />
- Good programming skills in Microsoft C#;<br />
- Experience with Windows Forms and desktop-based applications;<br />
- Familiarity with Object-Orientated Programming and design, including concepts such as interfaces, inheritance and class design;<br />
- Familiarity with Microsoft SQL Server and the retrieval of data from databases using .NET datasets and other data manipulation techniques;<br />
- Experience in manipulation of classes to display and edit data on Windows Forms;<br />
- Familiarity with Microsoft Office suite of programs to manipulate and populate MS Outlook, Word and Excel programmatically;<br />
- Knowledge of using version control software such as Git;<br />
- Knowledge of using Microsoft Office development tools;<br />
- Knowledge of using 3rd party components and their use in enhancing a development environment (e.g. DevExpress components);<br />
- Certain knowledge of Web API.
  </p>
